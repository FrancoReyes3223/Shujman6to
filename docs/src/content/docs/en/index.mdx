## Objective

Schujman<span style="color: var(--b2b-color)">**B2B**</span> is a web management application for employees and products. It allows users to securely authenticate and access a dashboard where they can manage staff and a product catalog for an organization.

## How to use it

1. **Log in** — enter your email and password on the home screen.
2. **Dashboard** — once authenticated, you access the main panel with three sections: overview, employees, and products.
3. **Manage employees** — from the *Employees* tab you can view, add, and delete staff records.
4. **Manage products** — from the *Products* tab you can view, add, and delete catalog products.
5. **Log out** — use the logout button in the sidebar to exit securely.
