## What is this project about?

A full-stack web application with user authentication. It includes a **Next.js** frontend, an **Express** backend with a layered architecture, and a **PostgreSQL** database managed with Prisma ORM.

## Tech stack

| Layer     | Technology                  |
|-----------|-----------------------------|
| Frontend  | Next.js 15 (App Router)     |
| Backend   | Node.js + Express           |
| ORM       | Prisma                      |
| Database  | PostgreSQL (Docker)         |
| Auth      | JWT + cookies               |
