## Objetivo

Schujman<span style="color: var(--b2b-color)">**B2B**</span> es una aplicación web de gestión para empleados y productos. Permite a los usuarios autenticarse de forma segura y acceder a un panel de control desde donde pueden administrar el personal y el catálogo de productos de una organización.

## ¿Cómo se usa?

1. **Iniciar sesión** — ingresá tu email y contraseña en la pantalla de inicio.
2. **Dashboard** — una vez autenticado, accedés al panel principal con tres secciones: resumen, empleados y productos.
3. **Gestionar empleados** — desde la pestaña *Empleados* podés ver, agregar y eliminar registros de personal.
4. **Gestionar productos** — desde la pestaña *Productos* podés ver, agregar y eliminar productos del catálogo.
5. **Cerrar sesión** — usá el botón de logout en el sidebar para salir de forma segura.
