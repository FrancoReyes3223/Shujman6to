## ¿De qué trata el proyecto?

Aplicación web completa con autenticación de usuarios. Incluye un frontend en **Next.js**, un backend en **Express** con arquitectura en capas, y una base de datos **SQLite** manejada con Prisma ORM.

## Stack tecnológico

| Capa       | Tecnología                  |
|------------|-----------------------------|
| Frontend   | Next.js 15 (App Router)     |
| Backend    | Node.js + Express           |
| ORM        | Prisma                      |
| Base datos | SQLite                      |
| Auth       | JWT + cookies               |
